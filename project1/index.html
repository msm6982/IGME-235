Projects

Romeo's Climb

  • 2D score based platformer that was developed in Unity for ScoreSpace Jam #9.
  • The prompt of the jam was "climbing" and using that prompt I came up with a game concept based on the dangerous trial mountain goats take to scale mountains.
  • I designed the platform challenge and camera position to feel daunting to reflect this ominous task.
  • I worked with two other artists to create the visuals of the project and another programmer to help with other game implementations.
Play Now!

Calamity Witch

  • ​ Top-down Arcade Survival game which was developed via MonoGame C# framework and designed with the given prompt of playing as Clumsy Witch.
  • Considering the prompt, I designed one of the prominent game mechanics being the three elemental spells the user wields and the three-element based monster types, and using a certain spell on a specific monster type can either destroy that monster or enrage them.
  • It makes users feel really clumsy when they use the wrong spell!
  • In addition to this mechanic, I implemented via C# coding: player movement, enemy and player spell casting, enemy AI, map object generation.
  • I worked with four other people on Calamity Witch.

Witch Doctor

  • I designed sprites and animations and created background lore for a basic 2D platformer that I based on classic fantasy and my hometown New Orleans.
