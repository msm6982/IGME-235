Prototype Design and Inspirations:

  • HTML and CSS implementation based on the implementation I used to build my Design to spec homework form a week prior, with the main differences being, utilizing grid box for the site and content layout instead of using flexbox, to achieve a more dynamic look.

  • I gain inspiration for the content layout and organization based on a few publicly hosted IGME student portfolios. I liked the contrasting background colors as different sections of the website’s content, it’s stylized yet professional looking.

  • I liked the retro-like yellow color I used for my initial background, so I made button-shaped borders for links which further sold the retro aesthetic and reminded me of arcade cabinets

  • All content sections have a 20vw padding on the left and right sides so content isn’t against the edge of the page. (10vw for mobile)
